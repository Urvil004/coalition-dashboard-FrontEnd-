## ✨ Features

| Feature | Description |
|---|---|
| 📊 **Blood Pressure Charts** | Interactive Chart.js line charts — systolic & diastolic over 6 months |
| 👥 **Patient Management** | Dynamic patient list with one-click profile switching |
| 📈 **Diagnosis History** | Comprehensive table with color-coded status indicators |
| 👤 **Patient Profiles** | DOB, gender, emergency contacts, and insurance info |
| 🧪 **Lab Results** | Downloadable lab result listings |
| 📱 **Fully Responsive** | Desktop, tablet, and mobile — mobile-first CSS |
| 🔗 **Live API Integration** | Real patient data from Coalition Technologies REST API |
| 🎨 **Modern UI** | Horizontal nav, glassmorphism design, smooth animations |

---

## 🚀 Tech Stack

```
Frontend  →  HTML5 · CSS3 (Grid / Flexbox) · Vanilla JavaScript
Charts    →  Chart.js v4.4.1
API       →  REST API with Basic Auth
Design    →  Manrope Font · Custom SVG Icons
Responsive → Mobile-first CSS Media Queries
```

## 🛠 Development Notes

- ⚡ **Zero Dependencies** — Chart.js CDN only
- 🧱 **Vanilla JS** — No frameworks, no build tools
- 📐 **CSS Grid / Flexbox** — Modern layout techniques
- 🖼 **Custom SVG Icons** — Lightweight & scalable
- 🚀 **Performance Optimized** — Minimal DOM updates
- ♿ **Accessibility Ready** — Semantic HTML, ARIA labels
